Accueil geOrchestra¶

Le projet geOrchestra a pour objectif de développer une Infrastructure de Données Spatiales modulaire, interopérable et Open Source.

Il intègre les meilleurs composants cartographiques du web dans une Infrastructure de Données Spatiales riche en fonctionnalités, libre, interopérable et facilement réutilisable. Référencé par l’Adullact, geOrchestra propose une solution clef en main pour publier et partager vos données localisées sur intranet et internet. Son code source est disponible et sous licence libre.

geOrchestra propose ainsi :

  • un visualiseur ;
  • un catalogue ;
  • un extracteur ;
  • un geopublisher ;
  • un site éditorial.

L’ensemble de ces modules sont indépendant et peuvent être installé seul ou ensemble. La page Aperçue des fonctionnalités vous donner une présentation rapide et visuel de geOrchestra.

La section Documentation vous donnera toutes les informations nécessaires quant à l’installation et la configuration des modules geOrchestra.

Vous trouverez de l’aide sur la liste de diffusion communautaire, section Communauté.

Le suivie du développement peut être réalisé via la forge Redmine prévue à cet effet.

Enfin tenez vous au courant des évolutions du projet via le blog.
